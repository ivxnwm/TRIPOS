# Groups

## Basics

**Definition 1.1** Binary operation
> A binary operation on a set $X$ is a function $*: X \times X \to X$.

**Definition 1.2** Group
> A group is a triple $(G, *, e)$ where
> * $G$ is a set,
> * $*$ is a binary operation on $G$
> * $e \in G$
> 
> satisfying 4 axioms:
> * closure :  $\forall a, b \in G$, $a * b \in G$
> * associativity : $\forall a, b, c \in G$, $(a * b) * c = a * (b * c)$
> * (right) identity : $\forall a \in G$, $a * e = a$
> * (right) inverses : $\forall a \in G$, $\exists b \in G$ such that $a * b = e$

**Proposition 1.3**
> Let $(G, *, e)$ be a group, let $a, b, b', e' \in G$. Then:
> * Right inverses are left inverses: $a * b = e \implies b * a = e$
> * Right identity is left identity: $e * a = a$
> * Inverses are unique: if $a * b = e$ and $a * b' = e$ then $b = b'$
> * Identity is unique: $a * e' = a \implies e' = e$
> 
> Remark: We can therefore write $a^{-1}$ for the unique inverse of $a$.

**Definition 1.4** Abelian group
> A group $(G, *, e)$ is abelian if $\forall a, b \in G$, $a * b = b * a$.

**Definition 1.5** Order of a group
> The order of a group $(G, *, e)$ is the number of elements of the set $G$, denoted $|G|$ or $\#G$.
> If $|G|<\infty$ then $G$ is finite, otherwise infinite.

**Definition 1.6** Bijection and permutation
> Let $X, Y$ be sets. A bijection is a map $f: X \to Y$ that has an inverse $g: Y \to X$ such that $f \circ g = id_Y$
> and $g \circ f = id_X$.
> 
> A bijection $f: X \to X$ is called a permutation of $X$.

**Definition 1.7** Symmetric group
> The symmetric group, $\textrm{Sym}(X)$, is the group of all permutations of a set $X$ under composition.
> If $X=\{1, 2, \ldots, n\}$, we write $S_n$ for $\textrm{Sym}(X)$.

**Definition 1.8** Subgroup
> Let $G$ be a group and $H \subseteq G$. Then $H$ is a subgroup of $G$ if:
> * $e \in H$
> * $\forall a, b \in H$, $a * b \in H$
> * $\forall a \in H$, $a^{-1} \in H$
> 
> We write $H \leq G$.
> 
> Remarks:
> * if $H \leq G$ then $H$ is a group
> * every group is a subgroup of itself
> * for any group $G$, $1_G=\{e_G\} \leq G$ is called the trivial subgroup
> * other subgroups are called proper subgroups

**Proposition 1.9**
> If $H\leq \mathbb{Z}$ then $H = n\mathbb{Z}$ for some $n \in \mathbb{N}$.

**Definition 1.10** Generators
> Let $X \subseteq G$, $G$ is a group.
> 
> $\langle X \rangle := \bigcap_{X \subseteq H \leq G} H$ is the subgroup of $G$ generated by $X$.
> 
> If $G = \langle X \rangle$ — say $X$ generates $G$ or or $X$ is a generating set of $G$.
> 
> Intuitively, $\langle X \rangle$ is the smallest subgroup of $G$ containing $X$.

**Definition 1.11** Isometry
> FOr any subset $X \subseteq \mathbb{C}$, an isometry of $X$ is a bijection $f: X \to X$ that preserves distances:
> $\forall x, y \in X$, $|f(x) - f(y)| = |x - y|$.

**Proposition 1.12** Isometry groups in $\mathbb{C}$
> Let $X \subseteq \mathbb{C}$. Then the set of isometries of $X$, $Isom(X)$, is a subgroup of $\textrm{Sym}(X)$.

**Definition 1.13** Dihedral groups
> Let $X_n \subseteq \mathbb{C}$ be the n-gon with vertices $\{e^{2\pi i k/n} : k=0, 1, \ldots, n-1\}$, $n \geq 3$.
> $D_2n := \textrm{Isom}(X_n)$ is the $n$th dihedral group.

**Lemma 1.14** Kite lemma
> Let $x_1, x_2, y_1, y_2 \in \mathbb{C}$. If $|y_1 - x_1| = |y_2 - x_1|$ and $|y_1 - x_2| = |y_2 - x_2|$ then
> $(x_2-x_1)
> \perp (y_2 - y_1)$.

**Lemma 1.15** 3-point lemma
> Let $X \subseteq \mathbb{C}$ and $f \in Isom(X)$. If $x_1, x_2, x_3 \in X$ are not collinear and $f(x_i) = x_i$ for
> $i=1, 2, 3$ then $f = id_X$.

**Theorem 1.16**
> $r(z) := e^{2\pi i/n} z$ (a rotation) and $s(z) := \overline{z}$ (a reflection).
> 
> $|D_{2n}| = 2n$ and $D_{2n} = \{e, r, r^2, \ldots, r^{n-1}, s, sr, sr^2, \ldots, sr^{n-1}\}$.
> 
> In particular, $\{r, s\}$ generates $D_{2n}$.

**Lemma 1.17** Dihedral relation
> For $r,s \in D_{2n}$ as above, $s r = r^{-1} s$.

**Definition 1.18** Homomorphism
> A map between groups $\varphi: G \to H)$ is called a homomorphism if $\forall g, g' \in G$,
> $\varphi(g * g') = \varphi(g) \cdot \varphi(g')$.
> 
> Remarks:
> * for any two groups $G, H$, the map $'vaarphi: G \to H$ defined by $\varphi(g) = G \to H$ given by $g \mapsto e_H$ is
> the trivial homomorphism.
> * if $H \leq G$ then the map $\varphi: H \to G$ given by $h \mapsto h$ is the inclusion homomorphism.

**Lemma 1.19** Homomorphism properties
> If $\varphi: G \to H$ is a homomorphism then:
> * $\varphi(e_G) = e_H$
> * $\forall g \in G$, $\varphi(g^{-1}) = \varphi(g)^{-1}$

**Definition 1.20** Isomorphism
> If a homoomorphism $\varphi: G \to H$ is also a bijection, then $\varphi$ is called an isomorphism and we write
> $G \cong H$.
> 
> From the perspective of group theory $G$ and $H$ are the same.

**Lemma 1.21**
> * If $\varphi$ is an isomorphism then so is $\varphi^{-1}$.
> * If $\varphi: G \to H$ and $\psi: H \to K$ are isomorphisms then so is $\psi \circ \varphi: G \to K$.
> * $\cong$ is an equivalence relation.

**Definition 1.22** Image and kernel
> Let $\varphi: G \to H$ be a homomorphism.
> * The image of $\varphi$ is $\textrm{Im}(\varphi) := \{h \in H: \exists g \in G$ such that $\varphi(g)=h \}$
> * The kernel of $\varphi$ is $\textrm{Ker}(\varphi) := \{g \in G: \varphi(g) = e_H \}$

**Proposition 1.23**
> Let $\varphi: G \to H$ be a homomorphism. Then:
> * $\textrm{Im}(\varphi) \leq H$
> * $\textrm{Ker}(\varphi) \leq G$

**Definition 1.24** Cyclic group
> A group $G$ is cyclic if $\exists g \in G$ such that $G = \langle g \rangle = \{g^k : k \in \mathbb{Z}\}$.

**Theorem 1.25**
> If $G$ is cyclic then $G \cong \mathbb{C}_n$ for some $n \in \mathbb{N}$ or $G \cong \mathbb{Z}$.

**Definition 1.26** Order of an element
> For any group $G$ and $g \in G$, $\langle g \rangle = \{g^k: k \in \mathbb{Z} \}$ is cyclic, so $\langle g \rangle
> \cong \mathbb{C}_n$ for some $n \in \mathbb{N} \cup \{\infty\}$. This $n$ is called the order of $g$, denoted
> $\textrm{o}(g)=|g|$.

**Lemma 1.27**
> Let $G$ be a group and $a, b \in G$ such that:
> * $aa^n=e$ for some $n \in \mathbb{N}$, $3n \geq 3$
> * $b^2=e$
> * $ab = ba^{-1}$
> 
> Then $\varphi(r^k) = a^k$, $\varphi(r^ks) = a^k b$ defines a homomorphism $\varphi: D_{2n} \to G$.

**Proposition 1.28**
> Suppose $G$ has generating set $\{a, b\}$ such that:
> * $a^n = e$ for some $n \in \mathbb{N}$, $3n \geq 3$
> * $b^2 = e$
> * $a^kb = ba^{-k}$
> * $|G| = 2n$
> 
> Then $G \cong D_{2n}$.

## Lagrange's theorem

**Definition 2.1** Left coset
> Let $H \leq G$ and $g \in G$. The corresponding left coset is $gH := \{gh: h \in H\} \subseteq G$.
> 
> The set of all left cosets of $H$ in $G$ is $G/H := \{gH: g \in G\}$.
> 
> The definition of right coset and $H \backslash G$ is similar.
> 
> The index of a subgroup $H \leq G$ is $[G:H := |G/H|$.

**Lemma 2.2**
> If $H \leq G$, the left cosets partition $G$, that is:
> * $G = \bigcup_{gH \in G/H} gH$ and
> * $\forall g_1H, g_2H \in G/H$, $g_1H \cap g_2H \neq \emptyset \implies g_1H = g_2H$.

**Lemma 2.3**
> Let $H \leq G$. Then $\forall g \in G$, there is a bijection $H \mapsto gH$.

**Theorem 2.4** Lagrange's theorem
> If $|G| < \infty$ then $|G| = [G:H] |H|$ for any $H \leq G$.

**Corollary 2.5**
> If $|G| < \infty$ and $g \in G$ then $o(g) \mid |G|$.

**Corollary 2.6**
> If $|g| < \infty$ and $g \in G$ then $g^{|G|} = e$.

**Corollary 2.7**
> If $|G|$ is prime then $G \cong \mathbb{C}_p$.

**Theorem 2.8** Fermat-Euler theorem
> Let $x, n \in \mathbb{N}$. IF $\gcd(x, n) = 1$, then $x^{\varphi(n)} \equiv 1 \mod n$.

## Group actions

**Definition 3.1** Group action
> An action of a group $G$ on a set $X$ is a map $G \times X \to X$ given by $(g, x) \mapsto g \cdot x$ such that:
> * $\forall x \in X$, $e \cdot x = x$
> * $\forall g, h \in G$, $\forall x \in X$, $g \cdot (h \cdot x) = (gh) \cdot x$.
> 
> We write $G \curvearrowright X$ to denote that $G$ acts on $X$.
> 
> Remarks:
> * $\forall G, X, gx=x$ defines the trivial action
> * $\textrm{Sym}(X) \curvearrowright X$
> * If $G \curvearrowright X$ and $H \leq G$ then $H \curvearrowright X$ by restriction
> * Every group $G$ acts on itself by $g \cdot x = gx$ — the left regular action.

**Theorem 3.2**
> An action of a group $G$ on a set $X$ defines a homomorphism $\varphi: G \to \textrm{Sym}(X)$.

**Theorem 3.3** Cayley's theorem
> Every group $G$ is isomorphic to a subgroup of a symmetric group. Furthermore, if $|G| < \infty$ then we may chose
> $\#X = |G|$.

**Definition 3.4** Orbit and stabilizer
> Let $G \curvearrowright X$, $x \in X$. Then:
> * The orbit of $x$ is $G x := \{y \in X: \exists g \in G$ such that $y=gx\} \subseteq X$
> * The stabilizer of $x$ is $G_x = \textrm{Stab}_G(x) := \{g \in G: gx = x\} \subseteq G$

**Definition 3.5** Transitive and faithful actions
> * An action $G \curvearrowright X$ is transitive if $\forall x, y \in X$, $\exists g \in G$ such that $gx = y$
> * An action $G \curvearrowright X$ is faithful if $\forall g \in G$, $g \neq e$, $\exists x \in X$ such that $gx \neq$ 
> $x$. Equivalently, $gx = hx$ $\forall x \in X \implies g=h$, or the homomorphism $\varphi: G \to \textrm{Sym}(X)$
> defined by the action is injective.

**Proposition 3.6**
> Suppose $G \curvearrowright X$. Then
> * $\forall x \in X$, $\textrm{Stab}_g(x) \leq G$
> * the orbits $\{Gy: y \in X \}$ partition $X$.
> 
> Remark:
> $Gx = X$ if and only if there is only one orbit — that is, the action is transitive.

**Theorem 3.7** Orbit-stabilizer theorem
> Suppose $G \curvearrowright X$ and $x \in X$. Then the map $g\textrm{Stab}_G(x) \mapsto gx$ defines a well-defined
> bijection $G/\textrm{Stab}_G(x) \to Gx$.

**Corollary 3.8**
> If $G \curvearrowright X$ then $\forall x \in X$, $|G| = |Gx| |\textrm{Stab}_G(x)|$.

**Theorem 3.9** Cauchy's theorem
> If $|G| < \infty$ and $p$ is a prime dividing $|G|$ then $\exists g \in G$ such that $o(g) = p$.

**Definition 3.10** Conjugates
> Let $G$ be a group. For any $g, h \in G$, the conjugate of $g$ by $h$ is $hgh^{-1}$.
> 
> Remark: If $G$ is abelian then $hgh^{-1} = g$ for all $g, h \in G$.

**Definition 3.11** Conjugacy class
> The conjugacy class of an $g \in G$ is $ccl(g) := \{hgh^{-1}: h \in G\} \subseteq G$.

**Definition 3.12** Centeraliser
> The centraliser of $g \in G$ is $C_G(g) := \{h \in G: hgh^{-1} = g\}$.
> 
> Remarks:
> * this is the stabilizer of $g$ under the conjugation action
> * $hgh^{-1} = g \iff hg = gh$ so the centraliser is the set of elements that commute with $g$.

**Definition 3.13** Center
> The center of a group $G$ is $Z(G) := \{g \in G: \forall h \in G, gh = hg\} = \bigcap_{g \in G} C_G(g)$.
> 
> This is the set of elements that commute with everything.

### From example sheets

**Proposition 3.14**
> If two elements are conjugate then they have the same order.

**Proposition 3.15**
> There is a bijection between the set of left cosets and the set of right cosets of a subgroup $H \leq G$.

**Proposition 3.16**
> If a $G$ contains elements of order $n$ and $m$ then $|G| \geq \textrm{lcm}(n, m)$.

**Proposition 3.17**
> If $K \leq H \leq G$ then $[G:K] = [G:H][H:K]$.

**Theorem 3.18**
> If $G \curvearrowright X$ then:
> * $\forall x \in X, h \in G$, $\textrm{Stab}_G(hx) = h \textrm{Stab}_G(x) h^{-1}$
> * $\forall h \in G$, $Fix_X(hgh^{-1}) = h Fix_X(g)$ where $Fix_X(g) := \{x \in X: gx = x\}$.

**Proposition 3.19**
> If $G \curvearrowright X$, $|G| < \infty$ and $G$ is abelian, the action is faithful and transitive, then $|G| = |X|$.

**Theorem 3.20** Burnside's lemma
> If $G \curvearrowright X$ and $|G| < \infty$ then the number of orbits of the action is given by
> $$\frac{1}{|G|} \sum_{g \in G} |Fix_X(g)|$$

**Corollary 3.21**
> If the action $G \curvearrowright X$ is transitive and $|X| > 1$ then $\exists g \in G$, $g \neq e$ such that
> $|Fix_X(g)| \geq 1$.

## Mobius group

**Definition 4.1** Riemann sphere
> The Riemann sphere is $\mathbb{C}_{\infty} := \mathbb{C} \cup \{\infty\}$ where $\infty$ is a formal point.

**Definition 4.2** Mobius transformation
> Let $a, b, c, d \in \mathbb{C}$ such that $ad - bc \neq 0$. If $c /neq 0$, the corresponding Mobius transformation
> $\mu: \mathbb{C}_{\infty} \to \mathbb{C}_{\infty}$ is defined by
> $$z \mapsto \begin{cases} \frac{az+b}{cz+d}, & \mbox{if } n \in \mathbb{C} \backslash \{ \frac{c}{d} \} \\ \infty, & \mbox{if } z = -\frac{c}{d} \\ \frac{a}{c}, & \mbox{if } z=\infty \end{cases}$$
> 
> If $c=0$, $\mu$ is defined by
> $$z \mapsto \begin{cases} \frac{az+b}{d}, & \mbox{if } z \in \mathbb{C} \\ \infty, & \mbox{if } z = \infty \end{cases}$$

**Theorem 4.3** Mobius group
> $\mathcal{M} := \{f: \mathbb{C}_{\infty} \to \mathbb{C}_{\infty}: f \mbox{ is a Mobius transformation} \}$ together
> with composition is a group called the Mobius group.

**Definition 4.4** Fixed point
> Let $f: X \to X$ be a permutation. A fixed point of $f$ is an $x \in X$ such that $f(x) = x$.

*Lemma 4.5** 3-point lemma for Mobius group
> If $\mu \in \mathcal{M}$ fixes 3 distinct points of $\mathbb{C}_{\infty}$ then $\mu = id.

**Proposition 4.6**
> Every $\mu \in \mathcal{M}$ has at least 1 fixed point.

**Lemma 4.7** Triple transitivity
> For any triples of distinct points $z_1, z_2, z_3 \in \mathbb{C}_{\infty}$ and $w_1, w_2, w_3 \in
> \mathbb{C}_{\infty}$, there exists $\mu \in \mathcal{M}$ such that $\mu(z_i) = w_i$ for $i=1, 2, 3$.
> 
> We say $\mathcal{M} \curvearrowright \mathbb{C}_{\infty}$ is sharply triply transitive.

**Definition 4.8** Cross ratio
> Let $z_1, z_2, z_3, z_4 \in \mathbb{C}_{\infty}$ be distinct. Since $\mathcal{M} \curvearrowright \mathbb{C}_{\infty}$
> is sharply triply transitive, there exists a unique $\alpha \in \mathcal{M}$ such that $\alpha(z_1) = 0$,
> $\alpha(z_2) = 1$, $\alpha(z_3) = \infty$.
> 
> The cross ratio of the quadruple is
> $$[z_1, z_2; z_3, z_4] := \alpha(z_4) = \frac{(z_1 - z_3)(z_2 - z_4)}{(z_1 - z_4)(z_2 - z_3)}$$

**Proposition 4.9** Generating set for Mobius group
> $\mathcal{M}$ is generated by the set of maps
> * $\alpha_a(z) := az$, $a \in \mathbb{C} \backslash \{0\}$
> * $\beta_b(z) := z + b$, $b \in \mathbb{C}$
> * $\gamma(z) := \frac{1}{z}$

**Definition 4.10** Circle
> A circle in $\mathbb{C}_{\infty}$ is either:
> * a Euclidean circle in $\mathbb{C}$
> * $l \cup \{\infty\}$ where $l$ is a Euclidean straight line in $\mathbb{C}$.

**Theorem 4.11** Mobius transformations preserve circles
> If $C \subseteq \mathbb{C}_{\infty}$ is a circle and $\mu \in \mathcal{M}$ then $\mu(C)$ is also a circle.

**Corollary 4.12**
> For points $z_1, z_2, z_3, z_4 \in \mathbb{C}_{\infty}$ lie on a circle if and only if $[z_1, z_2; z_3, z_4] \in
> \mathbb{R} \cup \{\infty\}$.

> The Mobius group is generated by the set of maps
> * $\beta_b(z) := z + b$, $b \in \mathbb{C}$
> * $\gamma(z) := \frac{1}{z}$

## Finite groups

**Definition 5.1** Direct product
> If $G$ and $H$ are groups, their direct product is $G \times H := \{(g, h): g \in G, h \in H\}$ with the
> group operation $(g_1, h_1)(g_2, h_2) := (g_1 g_2, h_1 h_2)$.
> 
> Remarks:
> * $(e_G, e_H)$ is an identity
> * $(g, h)^{-1} = (g^{-1}, h^{-1})$

**Theorem 5.2** Direct product theorem
> If $H_1, H_2 \leq G$ and
> * $H_1 \cap H_2 = \{e\}$
> * $\forall h_1 \in H_1$, $\forall h_2 \in H_2$, $h_1 h_2 = h_2 h_1$
> * $G = H_1 H_2 := \{h_1 h_2: h_1 \in H_1, h_2 \in H_2\}$
> * 
> then $G \cong H_1 \times H_2$.
> 
> Remark: If $H_1 \cap H_2 = \{e\}$ then $|H_1 H_2| = |H_1| |H_2|$. In particular, if $|G| = |H_1| |H_2|$ then the first
> condition implies the third.

**Lemma 5.3** Groups of order 4
> If $|G| = 4$ then either $G \cong \mathbb{C}_4$ or $G \cong K_4 \cong \mathbb{C}_2 \times \mathbb{C}_2$.

**Theorem 5.4** Chinese remainder theorem
> If $gcd(m, n) = 1$ then $\mathbb{C}_m \times \mathbb{C}_n \cong \mathbb{C}_{mn}$.

**Lemma 5.5** Groups of order 6
> If $|G| = 6$ then either $G \cong \mathbb{C}_6$ or $G \cong D_6$.
> 
> In particular, $S_3 \cong D_6$.

**Lemma 5.6** Groups of order 8
> If $|G| = 8$ then $G$ is isomorphic to one of the following:
> * $\mathbb{C}_8$
> * $\mathbb{C}_4 \times \mathbb{C}_2$
> * $\mathbb{C}_2 \times \mathbb{C}_2 \times \mathbb{C}_2$
> * $Q_8$

### From example sheets

**Proposition 5.7** Groups of order 2p
> Let $p$ be a prime and let $G$ be a group of order $2p$. Then either $G \cong \mathbb{C}_{2p}$ or $G \cong D_{2p}$.

**Proposition 5.8** Groups of order p^2
> Let $p$ be a prime and let $G$ be a group of order $p^2$. Then either $G \cong \mathbb{C}_{p^2}$ or $G \cong
> \mathbb{C}_p \times \mathbb{C}_p$.

## Quotients

**Definition 6.1** Normal subgroup
> $H \leq G$ is normal if $\forall h \in H, g\ in G$, $ghg^{-1} \in H$.
> 
> We write $H \triangleleft G$.
> 
> Remarks:
> * $\forall G$, $1 \triangleleft G$ and $G \triangleleft G$
> * if $G$ is abelian then every subgroup is normal
> * for any homomorphism $\varphi: G \to H$, $\textrm{Ker}(\varphi) \triangleleft G$

**Proposition 6.2**
>> Any subgroup of index 2 is normal.

**Lemma 6.3**
> If $H \leq G$ then $H \triangleleft G \iff \forall g \in G$, $gH = Hg$.

**Theorem 6.4** Quotient group
> If $H \triangleleft G$ then the set of (left) cosets $G/H$ is a group with operation
> $(g_1H)(g_2H) = (g_1 g_2)H$.

**Definition 6.5** Quotient group
> If $H \triangleleft G$ then the group $G/H$ defined in Theorem 6.3 is called the quotient group of $G$ by $H$.
> 
> Remarks:
> * $G/1 \cong G$ and $G/G \cong 1$
> * $\mathbb{Z}/n\mathbb{Z} \cong \mathbb{C}_n$
> * Beware! For groups $A, B, C$, $A/B \cong C \not\implies A \cong B \times C$.

**Theorem 6.6** First isomorphism theorem
> If $\varphi: G \to H$ is a homomorphism then $G/\textrm{Ker}(\varphi) \cong \textrm{Im}(\varphi)$.

**Definition 6.7** Simple group
> A group $G$ is simple if $|G| > 1$ and the only normal subgroups are $1$ and $G$ itself. Thus every homomorphism
> $G \to H$ is either trivial or injective.
> 
> Remark: $C_p$ is simple whenever $p$ is prime.

### From example sheets

**Proposition 6.8**
> A subroup $K \leq G$ is normal if and only if it is the kernel of some homomorphism.

**Proposition 6.9**
> Let $G$ be a finite group and $H \leq G$ a proper subgroup. If $|G|$ does not divide $|G:H|!$ then $H$ contains a
> nontrivial normal subgroup of $G$.

## Permuutations

**Definition 7.1** k-cycle
> Any list of k distinct elements $a_1, a_2, \ldots, a_k \in \{1, 2, \ldots, n\}$ defines a k-cycle
> $\sigma (a_1 a_2 \ldots a_k) \in S_n$ as follows:
> $\sigma(b) = \begin{cases} a_{i+1}, & \mbox{if } b = a_i \mbox{ for some } 1 \leq i < k \\ a_1, & \mbox{if } b = a_k
> \\ b, & \mbox{otherwise} \end{cases}$
> 
> Remark:
> $(a_1 a_2 \ldots a_k) = (a_2 a_3 \ldots a_k a_1) etc.

**Definition 7.2** Disjoint cycles
> Cycles $(a_1 a_2 \ldots a_k)$ and $(b_1 b_2 \ldots b_m)$ are disjoint if $a_i \neq b_j \forall i, k$.
> 
> Note that disjoint cycles commute.

**Theorem 7.3** Cycle decomposition
> Every $\sigma \in S_n$ can be written as a product of disjoint cycles. This expression is unique up to
> * shifting the elements in a cycle
> * reordering the cycles

**Definition 7.4** Cycle type
> If $\sigma = (a_1^1 a_2^1 \ldots a_{k_1}^1)(a_1^2 a_2^2 \ldots a_{k_2}^2) \ldots (a_1^l a_2^l \ldots a_{k_l}^l)$ is a
> product of disjoint cycles, then $\sigma$ is called a ($k_1, k_2, \ldots, k_l$)-cycle and the set $\{k_1, k_2, \ldots,
> k_l\}$ is called the cycle type of $\sigma$.
> 
> Remarks:
> * We often omit singletons — e.g. $(1 2)(3)(4 5)$ is a (2, 2)-cycle
> * If $\sigma$ is a $k$-cycle then $o(\sigma) = k$. More generally, if $\sigma$ has cycle type $(k_1, k_2, \ldots,
> k_l)$ then $o(\sigma) = \textrm{lcm}(k_1, k_2, \ldots, k_l)$.

**Definition 7.5** Transposition
> A transposition is a 2-cycle.
> A transposition of the form $(i \; i+1)$ is called adjacent.

**Theorem 7.6** Transpositions generate $S_n$
> The set of transpositions generates $S_n$ for any finite $n$.

**Lemma 7.7**
> Any transposition can be written as a product of an odd number of adjacent transpositions.
> In particular, $S_n$ is generated by adjacent transpositions.

**Lemma 7.8**
> If $\tau_1, \dots, \tau_k$ are transpositions such that $\sigma = \tau_1 \ldots \tau_k = e$ then $k$ is even.

**Theorem 7.9** Sign homomorphism
> The map $\textrm{sign}: S_n \to \{1, -1\}$ given by $\textrm{sign}(\tau_1 \ldots \tau_k) = (-1)^k$ where $\tau_i$ are
> transpositions is a well-defined homomorphism.
> 
> Remark: The cycle type makes it easy to determine the sign of a permutation.
> Indeed, $(a_1, \dots, a_k) = (a_1 a_k)(a_1 a_{k-1}) \ldots (a_1 a_3)(a_1 a_2)$ so $(a_1, \dots, a_k)$ is even if
> and only if $k$ is odd(!). More generally, a ($k_1, \ldots, k_l$)-cycle is even if and only if the number of even
> $k_i$ is even.

**Definition 7.10** Alternating group
> If $sign(\sigma) = 1$ then $\sigma$ is even, otherwise it is odd.
> The subgroup $A_n := \textrm{ker}(\textrm{sign}) \triangleleft S_n$ is called the alternating group on n elements.

**Theorem 7.11** Conjugacy in $S_n$
> Two permutations in $S_n$ are conjugate if and only if they have the same cycle type.

**Lemma 7.12** Conjugacy classes in $A_n$
> Let $\gamma \in A_n \triangleleft S_n$.
> * If some odd element commutes with $\gamma$ then $ccl_{A_n}(\gamma) = ccl_{S_n}(\gamma)$.
> * If every element of $S_n$ that commutes with $\gamma$ is even then $ccl_{A_n}(\gamma)$ splits into two:
> $ccl_{S_n}(\gamma) = ccl_{A_n}(\gamma) \cup ccl_{A_n}(\tau \gamma \tau^{-1})$ where $\tau$ is any transposition.

**Theorem 7.13** $A_5$ is simple
> $A_5$ is simple.
